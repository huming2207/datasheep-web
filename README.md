# DataSheep Web UI

Yet another Kanban system, for embedded software developers.

## Tech stacks

- Language: TypeScript
- Framework: 
    - Vue 2
    - Vue class-based components
    - Vuetify
    - Vuelidate
    - Axios

## Vue project setup 

### Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
